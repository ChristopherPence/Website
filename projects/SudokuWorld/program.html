How to use the Program

Generate a puzzle: In order to generate a puzzle you must first select a difficulty level from the menu in the top right. Then you hit the "generate" button to make a puzzle with that difficulty.

Solve a puzzle: In order to solve a puzzle you must first enter in a puzzle by hand or use one that the generator has supplied. Then you click the solve button and the program will solve the puzzle.

Clearing the board: This can simply be done by pressing the clear button in the top left.

Printing the board: This can simply be done by pressing File then Print. This will allow you to select your printer settings then print the board.
